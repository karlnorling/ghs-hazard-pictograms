# React Components

Inline SVG components for all 24 GHS and transport hazard pictograms. Each component is tree-shakeable, fully accessible, and accepts the same `PictogramProps` interface.

## Installation

```sh
npm install @ghs-pictogram/react
# or
yarn add @ghs-pictogram/react
```

## Basic usage

```tsx
import { Ghs01Explosive } from '@ghs-pictogram/react';

export default function Label() {
  return <Ghs01Explosive width={120} height={120} />;
}
```

## Props reference — `PictogramProps`

Every component accepts the following props. Use the controls panel below to experiment interactively.

<Canvas of={ReactStories.Explosive} />
<Controls of={ReactStories.Explosive} />

| Prop          | Type                  | Default                      | Description                                              |
| ------------- | --------------------- | ---------------------------- | -------------------------------------------------------- |
| `width`       | `number \| string`    | SVG intrinsic width          | Width of the `<svg>` element                             |
| `height`      | `number \| string`    | SVG intrinsic height         | Height of the `<svg>` element                            |
| `title`       | `string`              | Pictogram name               | Accessible title injected as `<title>` inside the SVG   |
| `description` | `string`              | Hazard description           | Accessible description injected as `<desc>`              |
| `className`   | `string`              | —                            | CSS class forwarded to the root `<svg>`                  |
| `style`       | `React.CSSProperties` | —                            | Inline styles forwarded to the root `<svg>`              |
| `aria-label`  | `string`              | —                            | Overrides `aria-labelledby` with a direct label          |

---

## Accessibility

Every component renders with `role="img"`, `aria-labelledby` pointing to an injected `<title>` and `<desc>` element. Screen readers announce the pictogram name and hazard description automatically.

Override the defaults when you need domain-specific copy:

<Canvas of={ReactStories.CustomA11y} />

```tsx
<Ghs06Toxic
  title="Danger: Toxic substance"
  description="Acutely toxic — wear PPE before handling."
  width={120}
/>
```
